Implementa una función llamada obtenerInformacion que reciba un parámetro llamado id que representa el identificador de un usuario en una API externa. La función debe hacer una llamada a la API externa para obtener la información del usuario con el identificador especificado en el parámetro id. Si la llamada a la API es exitosa, la función debe retornar una promesa que se resuelve con la información del usuario. Si la llamada a la API es fallida, la función debe retornar una promesa que se rechaza con un mensaje de error.
