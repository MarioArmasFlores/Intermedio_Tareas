"No habrá penitencia alguna que exoneré este pecado. El precio a pagar por la mayor de las blasfemias."

גִּבְעַת הָאֵפֶר שֶׁבְּלַעְהָ חֲטָאִים וּפוֹשְׁעִים לֹא חָמְלָה עֲלֵיכֶם גַּם. הַמִּילָגָה הָיִיתֶם וְעַל יְדֵי הַמִּילָגָה תִּשְׁאֲרוּ, לֹא תִּהְיוּ עוֹד אֶלָּא פָנִים אֲנוֹנִיּוֹת נוֹצָרוֹת, לֹא קוֹל וְלֹא מַבּוּע. עֲנִיַּתְכֶם הִסְתַּיְּמָה.
